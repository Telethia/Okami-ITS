# Okami-ITS
Program that reads ITS files from Okami and prints contents.

# How to Use
Open OkamiITS.exe and it will prompt you to drag a file into it. Drag the ITS file you want to see the contents of into the program, which will ask you if it's from the November Prototype or Final. Select which version it is (anything from the final PS2, Wii, or HD versions are considered 'Final') and it'll create a file in the directory of the original .ITS file with the contents.
